## 新功能
### 配置系统
只需轻轻一点, 就能更改JSON配置, 还能使用别人的配置.
每次加载脚本从cookie中直接读取配置, 无需每次设置.
```json
{
    "name": "Default",
    "autoAnswerEnabled": false,
    "defaultAnswer": "A",
    "autoSignInEnabled": true,
    "scanInternal": 5000,
    "autoFlowerEnabled": false,
    "autoRespeakEnabled": false,
    "autoRespeakTrigger": 3
}
```
